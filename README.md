# IIAS
## Для начала работы с программой :

```sh
git clone https://github.com/Mattakushii/iias_cval.git
git checkout develop
cd iias_cval
yarn
cd server
yarn
```

## Для запуска реакта и сервера:
```
//В папке iias_cval
yarn start
//Открыть новый терминал
cd server
yarn server
```
